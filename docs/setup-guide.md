# Local Setup Guide (Windows + XAMPP)

1. Install XAMPP from the official Apache Friends website.
2. Open XAMPP Control Panel and start **Apache** and **MySQL**.
3. Clone this repository.
4. Move the project folder into `C:\xampp\htdocs\`.
5. Open `http://localhost/phpmyadmin`.
6. Import `database/sql/init_databases.sql` in SQL tab.
7. Open `http://localhost/bookshop-management-system/` to run the project.
